A comprehensive, conversion-optimized website showcasing advanced digital marketing services with AI-powered automation, SEO optimization, and performance tracking.

### **Services Showcase**
- **6 Core Services** with detailed descriptions and pricing

### **SEO Automation System**
- **Technical SEO** - Meta tags, schema markup, sitemaps

## 🚀 Quick Start

### **Local Development**
1. Clone the repository:
   ```bash
   git clone https://github.com/PantMayank81/outta-the-box-marketing.git
   cd outta-the-box-marketing
   ```

2. Open `index.html` in your browser or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Using the included script
   chmod +x start-server.sh
   ./start-server.sh
   ```

3. Visit `http://localhost:8000` to view the website

### **Production Deployment**
- **GitHub Pages**: Enable in repository settings
- **Netlify**: Connect repository for automatic deployment
- **Vercel**: Deploy with zero configuration
- **Custom Server**: Upload files to your web server
